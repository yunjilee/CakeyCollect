To play the game:
1. Import cakeyScript.sql into MySQL and run it 
2. Run Server.java
3. Run Player.java
4. Click LOGIN/REGISTER
5. Type in a username and password and click Register
6. Click Login
7. Click Options
8. Choose your player
9. Play game
